# OT Cybersecurity Layered Context Architecture

This folder contains the domain, architecture, implementation, cybersecurity, and gantt contexts for the OT Cybersecurity TS type, following the UGS canonical architecture.

## Structure
- `domain_context.txt`: Reusable domain knowledge.
- `architecture_context.txt`: Architecture knowledge.
- `implementation_context.txt`: Implementation methodology, FAT, SAT, exclusions, etc.
- `cybersecurity_context.txt`: Cybersecurity guidance.
- `gantt_context.txt`: Implementation phases, milestones, draw.io guidance.
- `context.txt.template`: The Placeholder Registry template for new projects.
- `section_guidance/`: Contains guidance files for each specific TS section.

## Global Rules
1. Project metadata overrides all.
2. Saved section content overrides drafts and history.
3. Use placeholders; do not invent specific vendor names, IP addresses, asset counts unless confirmed.
