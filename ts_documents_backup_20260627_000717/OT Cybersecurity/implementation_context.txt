Implementation phases: M1 — Assessment & Design (~4w): OT risk assessment, asset discovery, security architecture design (FDS); M2 — Procurement & Configuration (~6w): Hardware/software procurement, firewall rulesets, EDR policies; M3 — FAT (~2w): Policy review, remote access test; M4 — Installation & Deployment (~4w): Firewall deployment, EDR rollout, NDR sensor tuning; M5 — Handover (~2w): Training, incident response playbook, sign-off. Total durations indicative.

FAT: Review of security policies, firewall rule sets (offline), and demonstration of remote access and monitoring capabilities.

Buyer obligations: Provide network diagrams, asset inventory (if available), downtime windows for deployment, and active directory integration details. Assign IT and OT stakeholders. Ensure necessary SPAN/mirror ports are configured on core switches.

Value addition: Cybersecurity awareness training session for OT personnel; initial vulnerability scan report.

Standard exclusions: IT enterprise security implementation; patching of obsolete/unsupported OS (e.g., Windows XP); physical security (cameras, access control); L1 code modifications.

Risks: Legacy equipment may not support modern EDR agents or authentication. Unplanned downtime during firewall insertion.
