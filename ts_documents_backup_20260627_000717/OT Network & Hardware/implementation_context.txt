Implementation phases: M1 — Engineering & Design (~6w): network architecture diagram, cable routing layout, bill of materials (BOM), IP addressing plan; M2 — Procurement (~12-16w): ordering and delivery of switches, servers, and cables; M3 — FAT (~2w): staging and configuration of network equipment offline; M4 — Installation & Cabling (~8w): physical mounting, cable pulling, termination, and splicing; M5 — Commissioning & Handover (~3w): power-up, ping tests, failover testing, network certification, as-built documentation. Total durations indicative.

FAT: Staging of core switches and servers to verify configuration, VLAN routing, and basic failover (if applicable).

Buyer obligations: Provide site access, civil works (trenching, conduits), clean UPS power drops at rack locations, environment control (HVAC) in server rooms, and existing IP addressing scheme. Assign a site supervisor for installation coordination.

Value addition: Network health assessment report upon commissioning; extended hardware warranty negotiation with OEMs.

Standard exclusions: Civil works (trenching, drilling); primary power supply/UPS; HVAC systems; structural modifications to plant; legacy equipment remediation; L1/L2 application software deployment.

Risks: Long lead times for specialized industrial hardware (e.g., Cisco IE switches). Cable routing delays due to plant operations or missing conduits.
