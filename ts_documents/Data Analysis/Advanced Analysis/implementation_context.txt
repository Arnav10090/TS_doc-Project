Total durations indicative.

FAT: Demonstration of the ML pipeline on a fixed historical dataset, validating that the model achieves the agreed-upon accuracy metrics.

Buyer obligations: Provide a clean, historical dataset (minimum 12-24 months of data depending on the use case) at project kickoff; provide SME domain experts to assist with feature engineering; ensure live data feeds are available for M3.

Value addition: Post-go-live model monitoring and retuning for the first 3 months.

Standard exclusions: Installing new physical sensors to fill data gaps discovered during M1; modifying L1 PLC logic to act automatically on predictions (this is an advisory system only).

Risks: Poor historical data quality (gaps, noise, incorrect timestamps) will significantly delay model development.
